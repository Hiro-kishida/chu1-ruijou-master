# 中1数学　累乗マスター

## 目的
中学1年生向けの、累乗・指数の計算練習アプリ。
スマホで取り組みやすいよう、電卓風UIで答えを入力する。

## ver0.2 で反映した内容
- 1回5問チャレンジ
- 問題は1問ずつ大きく表示
- 正解時だけ、やさしいピンポン音
- 不正解時は音なし・長い解説なし
- 全問正解なら紙吹雪つきのクリア演出
- 誤答があった場合は、△の問題だけ一覧表示
- △の問題だけ再挑戦
- 将来のSpreadsheet連携に備えて、内部では
  問題・入力答え・正答・初回／再挑戦・問題タイプを扱える設計

## 数学上の設計判断
- ステージ3「カッコあり／なし」は2乗問題のみ扱う
- 理由：
  (-2)² = 4、-2² = -4 で違いが明確に出るため
  3乗では答えが同じになり、比較練習に適さないため
